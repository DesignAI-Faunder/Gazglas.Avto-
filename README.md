# GasGlaz.AI

Мобильное PWA-приложение на React + Tailwind CSS в стиле Dark Techno.

## Что реализовано

- главный экран с виджетом качества воздуха (заглушка OpenWeather API);
- крупная кнопка запуска теста выхлопа;
- нижние кнопки: "Карта АЗС" и "История чеков";
- раздел "Тарифы" с ценами BYN / RUB в отдельном блоке констант;
- экран камеры с рамкой-прицелом;
- таймер теста 10 секунд;
- серия snapshot-снимков каждые 2 секунды;
- бегущие технологические логи ИИ;
- PWA: `manifest.json` и `service-worker.js`;
- заготовка Supabase Edge Function для серверного анализа;
- структура для загрузки PDF-справочников под RAG.

## Быстрый старт

1. Установите зависимости:

```bash
npm install
```

2. Создайте `.env` на основе `.env.example`.

3. Запустите приложение:

```bash
npm run dev
```

## Безопасность API-ключей

Ключ `OPENAI_API_KEY` должен храниться только в Supabase Edge Functions, а не во frontend-коде.
